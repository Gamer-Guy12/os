Or this shit together to make cool errors!

First 48 bits are for flags and the rest is just the error number

Bit 0: Too much space / KGFX wrong mode
Bit 1: Missing Block

first 48 bits for error 5:
Bit 0: IRQ Handling error
Bit 1: Keyboard handling error

first 48 bits for error 6:
Bit 0: Thread Self Termination
Bit 1: Process Self Termination

ERR 0: Memory Allocator Error
ERR 1: Multiboot Error
ERR 2: Memory Init Error
ERR 3: No interrupt handler
ERR 4: KGFX Error
ERR 5: HAL init error
ERR 6: Self Termination error
